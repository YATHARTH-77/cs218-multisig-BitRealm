## рҹ“– Project Overview

This repository contains a **secure, gas-optimized Smart Contract backend** and a **Next.js frontend** for an N-of-M Multi-Signature Wallet. The contract enables multiple authorized owners to manage funds collectively, requiring a predefined threshold of approvals before any transaction can be executed.

The backend is built using **Hardhat v3 (ESM)** and utilizes **Hardhat Ignition** for declarative, state-tracked deployments.

## рҹӣ пёҸ Option 2: Local Setup & Development

> This `main` branch is pre-configured for local machine evaluation вҖ” ideal for auditing mathematical security, gas optimizations, and full architecture walkthrough.

### Step 1 вҖ” Install Dependencies

```bash
npm install
```

### Step 2 вҖ” Compile the Contract

```bash
npx hardhat build
```

### Step 3 вҖ” Run the Test Suite

```bash
npx hardhat test
```

### Step 4 вҖ” View Gas Report

```bash
npx hardhat test --gas-stats
```

### Step 5 вҖ” View Coverage Report

```bash
npx hardhat test --coverage
```

### Step 6 вҖ” Deploy to Persistent Local Network

**Terminal 1** вҖ” Start the local node:
```bash
npx hardhat node
```

**Terminal 2** вҖ” Deploy the contract:
```bash
npx hardhat ignition deploy ignition/modules/MultiSigWallet.ts --network localhost
```

**Terminal 2** вҖ” Fund the contract:
```bash
npx hardhat run scripts/fund.ts --network localhost
```
